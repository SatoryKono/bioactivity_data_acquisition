# 31 Target UniProt QC

**Version:** 1.0.0
**Date:** 2025-01-29
**Author:** Data Acquisition Team

## Purpose

This document describes QC metrics and thresholds for the Target (UniProt) pipeline.

## QC Metrics

- Duplicate detection
- Accession format validation
- Taxonomy ID validation
- Annotation coverage

## QC Thresholds

- `duplicate_ratio`: 0.0 (critical)
- `accession_validity`: 1.0 (all must be valid)
