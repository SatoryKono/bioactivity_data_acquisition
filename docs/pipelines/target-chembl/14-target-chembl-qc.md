# 14 Target ChEMBL QC

**Version:** 1.0.0  
**Date:** 2025-01-29  
**Author:** Data Acquisition Team

## Purpose

This document describes QC metrics and thresholds for the Target (ChEMBL) pipeline.

## QC Metrics

- Duplicate detection
- UniProt accession coverage and validation
- Target type distribution
- Referential integrity

## QC Thresholds

- `duplicate_ratio`: 0.0 (critical)
- `uniprot_coverage`: Configurable threshold
