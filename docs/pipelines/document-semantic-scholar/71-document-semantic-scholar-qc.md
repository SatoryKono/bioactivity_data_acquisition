# 71 Document Semantic Scholar QC

**Version:** 1.0.0 **Date:** 2025-01-29 **Author:** Data Acquisition Team

## Purpose

This document describes QC metrics and thresholds for the Document (Semantic
Scholar) pipeline.

## QC Metrics

- Duplicate detection
- Identifier coverage (DOI/ArXiv)
- Citation count validation
- Abstract coverage

## QC Thresholds

- `duplicate_ratio`: 0.0 (critical)
- `identifier_coverage`: Configurable threshold
