# 39 Target IUPHAR QC

**Version:** 1.0.0 **Date:** 2025-01-29 **Author:** Data Acquisition Team

## Purpose

This document describes QC metrics and thresholds for the Target (IUPHAR)
pipeline.

## QC Metrics

- Duplicate detection
- ID format validation
- UniProt reference validation
- Family classification coverage

## QC Thresholds

- `duplicate_ratio`: 0.0 (critical)
- `id_validity`: 1.0 (all must be valid)
