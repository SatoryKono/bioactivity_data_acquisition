# 55 Document OpenAlex QC

**Version:** 1.0.0  
**Date:** 2025-01-29  
**Author:** Data Acquisition Team

## Purpose

This document describes QC metrics and thresholds for the Document (OpenAlex) pipeline.

## QC Metrics

- Duplicate detection
- Identifier coverage (DOI/PMID)
- Citation count validation
- Concept coverage

## QC Thresholds

- `duplicate_ratio`: 0.0 (critical)
- `identifier_coverage`: Configurable threshold
