# Onboarding Checklist

Этот документ помогает новым контрибьюторам быстро синхронизироваться с правилами именования.

- Ознакомьтесь с `docs/styleguide/00-naming-conventions.md` для правил документов и ссылок.
- Изучите `docs/styleguide/11-naming-policy.md` и актуальный [план нормализации имён](../styleguide/12-naming-normalization-plan.md) перед любыми переименованиями.
- Перед коммитом запускайте `bioetl-validate-naming-violations`; при появлении нарушений обновляйте `docs/styleguide/VIOLATIONS_TABLE.md` и указывайте дедлайны решения.
- Все временные исключения фиксируйте в PR и синхронизируйте с чек-листами плана нормализации.
